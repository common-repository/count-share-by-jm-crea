Intégrez des boutons de partage vers les réseaux sociaux sur vos pages et sur vos posts. Compteurs inclus.

== Description ==

Count Share créé par JM Créa, permet d’ajouter les boutons de partage vers les réseaux sociaux dans vos pages, articles et produits Woocommerce.


Ce plugin est entièrement paramétrable : 

- Possibilité d’afficher les boutons et de les masquer
- Choix du positionnement (haut, bas, haut et bas)
- Choix des endroits à afficher (pages, posts, produits woocommerce)
- Choix des couleurs de fond des boutons
- Choix des couleurs des textes des boutons
- Possibilité d’afficher et masquer les textes sur les boutons
- Possibilité de régler les bords arrondis des boutons (radius)
- Possibilité d’afficher et masquer les compteurs

Vous avez aussi la possibilité d’appeler vos boutons avec le shortcode suivant : [cs_by_jm_crea]

Vous remarquerez qu’il y a possibilité de réinitialiser le plugin si besoin avec un simple clic.

== Installation ==

1. Uploadez `count-share-by-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Count Share par JM Créa’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite dans l’onglet ‘Count Share’

1. Vous pouvez aussi appeler vos boutons avec le shortcode : [cs_by_jm_crea]
